# 채팅
채팅방을 생성하고, 생성된 채팅방 리스트를 확인할 수 있습니다. 카카오워크에서는 채팅방에 대한 다양한 관리 및 설정 기능을 제공합니다. 또한 채팅방 안에서 음성채팅, 메시지 검색, 멘션, 공지, 이모지 반응 등 유용한 기능들을 활용해 멤버들과 대화를 나눌 수 있습니다.

## 채팅 탭

**채팅** 탭에서는 채팅방을 생성하고, 생성된 채팅방 리스트를 확인할 수 있습니다. 카카오워크에서는 채팅방에 대한 다양한 관리 및 설정 기능을 제공합니다. 또한 채팅방 안에서 음성채팅, 메시지 검색, 멘션, 공지, 이모지 반응 등 유용한 기능들을 활용해 멤버들과 대화를 나눌 수 있습니다.

| ⓵ 채팅방 정렬   | 최신 메시지 순, 안 읽은 메시지 순으로 채팅방 정렬                                                                                                                |
| ⓶ 채팅방 생성   | <p>멤버, 조직명을 검색하거나 조직도를 통해 1:1 채팅방 또는 그룹 채팅방을 생성<br>- 관리자가 비밀 채팅방 사용 On 으로 설정한 경우, 일반 채팅방/비밀 채팅방 생성 옵션 제공</p>                                 |
| ⓷ 채팅방 리스트  | <p>참여 중인 모든 채팅방 표시<br>- Bot과의 채팅방, 나와의 채팅방, 그룹 채팅방, 비밀 채팅방 등</p>                                                                             |
| &nbsp;&nbsp;&nbsp;&nbsp;전체 채팅방   | <p>[All] 아이콘이 붙은 채팅방으로, 모든 워크스페이스 멤버들이 자동으로 참여하는 채팅방</p><ul><li>조직도와 연동되어 멤버 자동 참여 및 탈퇴 처리</li><li>모든 워크스페이스에 기본으로 생성되며 임의로 퇴장 불가능</li></ul> |
| &nbsp;&nbsp;&nbsp;&nbsp;나와의 채팅방 | <p>[나] 아이콘이 붙은 채팅방으로, 나만 참여 및 조회할 수 있는 채팅방</p><ul><li>임의로 퇴장 불가능</li></ul>                                                                   |
| &nbsp;&nbsp;&nbsp;&nbsp;그룹 채팅방   | 팀 별, 프로젝트 별 목적에 따라 생성하여 세 명 이상의 멤버로 구성된 채팅방                                                                                                  |
| &nbsp;&nbsp;&nbsp;&nbsp;1:1 채팅방 | 두 명의 멤버로 구성된 채팅방으로, 다른 멤버와 1:1로 대화하는 채팅방                                                                                                     |
| &nbsp;&nbsp;&nbsp;&nbsp;봇과의 채팅방 | 카카오워크 필수 기능 안내, 기기 인증 등 중요한 자동 알림 메시지를 수신하는 채팅방                                                                                              |
| &nbsp;&nbsp;&nbsp;&nbsp;그룹사 채팅방 | 파란색 채팅 아이콘이 붙은 채팅방으로, 그룹사 멤버가 참여한 채팅방                                                                                                        |
| &nbsp;&nbsp;&nbsp;&nbsp;외부 채팅방  | 빨간색 채팅 아이콘이 붙은 채팅방으로, 외부 워크스페이스 멤버가 참여한 채팅방                                                                                                  |
| &nbsp;&nbsp;&nbsp;&nbsp;비밀 채팅방   | 자물쇠 아이콘이 붙은 채팅방으로, 메시지가 자동으로 삭제되는 채팅방                                                                                                        |
